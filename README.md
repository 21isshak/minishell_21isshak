# My minishell (miniHELL)

This is a 42 project where we create our own minishell(mini version of bash).

This project is not finished.

There are some mistakes, possible non-allowed functions used and norminette is not corrected.

Check to_do_list to see for what improvements needs to be done.

## Installation and Testing

to install this project clone the [repo](https://github.com/21isshak/minishell_21isshak.git)

```bash
git clone https://github.com/21isshak/minishell_21isshak.git
```

then open the repo and do make

```bash
cd minishell
```

```bash
make
```

```bash
./minishell
```

now it is possible to test and work on minishell
